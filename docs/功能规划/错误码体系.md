# 错误码体系

## 目标

错误码用于把「用户能看懂的提示」和「开发排查需要的技术细节」分开。

用户界面只展示：

- 发生位置
- 人话提示
- 错误编号
- 原因说明
- 处理建议
- 是否可重试

后端日志保留：

- `source`
- `user_message`
- `technical_message`
- `error_code`
- `context_json`
- `created_at`

前端不得展示 SQL、堆栈、文件路径、系统异常原文。

## 分类原则

- 一个错误码只表达一种失败原因。
- 错误码按领域分组，便于后续模块接入。
- 用户提示稳定，技术日志可详细。
- 可重试错误不等于自动重试，自动重试必须由对应模块设计。
- 导入器后续必须复用本体系，不单独发明错误码格式。

## 错误码总表

| 错误码 | 领域 | 用户提示 | 常见原因 | 策略 |
| --- | --- | --- | --- | --- |
| `APP_UNKNOWN` | system | 操作没有完成，请稍后再试。 | 未归类异常 | 记录技术详情，后续补码 |
| `APP_INVALID_ARGUMENT` | system | 输入内容不完整，请检查后再试。 | 参数缺失、ID 不合法 | 阻止请求进入存储层 |
| `APP_CONFIG_INVALID` | system | 配置内容有误，请检查设置后再试。 | 表名、连接配置、契约配置非法 | 阻止启动相关数据层 |
| `MYSQL_UNAVAILABLE` | mysql | 数据库暂时连接不上，内容会先保存在本机。 | MySQL 不可用、连接中断、账号不可用 | 课程分区走本地兜底，强一致数据阻止写入 |
| `MYSQL_QUERY_FAILED` | mysql | 数据库写入没有完成，请稍后再试。 | SQL 执行失败、索引冲突、事务回滚 | 记录 MySQL 错误码 |
| `LOCAL_STORE_READ_FAILED` | local-store | 本机缓存读取失败，系统会尝试重新初始化。 | 本地 JSON 损坏、不可读 | 隔离损坏文件 |
| `LOCAL_STORE_WRITE_FAILED` | local-store | 本机缓存没有保存成功，请稍后再试。 | 磁盘权限、文件占用、原子写失败 | 记录文件错误 |
| `PENDING_OPERATION_INVALID` | local-store | 本机待同步记录异常，系统会保留现场。 | pending 队列结构不符合协议 | 停止重放，保留现场 |
| `PENDING_OPERATION_REPLAY_FAILED` | local-store | 部分本机改动还没同步到数据库。 | MySQL 恢复后重放失败 | 保留失败操作和后续操作 |
| `COURSE_NAME_EMPTY` | course | 知识库名称不能为空。 | 创建或重命名课程名称为空 | 前端提示补充 |
| `COURSE_NOT_FOUND` | course | 这个知识库不存在或已被删除。 | 课程已软删除或 ID 失效 | 刷新列表 |
| `COURSE_SAVE_FAILED` | course | 知识库保存没有完成，请稍后再试。 | 课程创建、移动、删除、旧保存失败 | 命令式操作可进入本地兜底 |
| `SECTION_NAME_EMPTY` | section | 分区名称不能为空。 | 分区名称为空 | 前端提示补充 |
| `SECTION_NAME_DUPLICATE` | section | 已经有同名分区了。 | 同名分区冲突 | 阻止写入 |
| `SECTION_NOT_FOUND` | section | 这个分区不存在或已被删除。 | 分区已软删除或 ID 失效 | 刷新列表 |
| `MINDMAP_REQUEST_INVALID` | mindmap | 导图请求不完整，请重新打开后再试。 | 缺课程 ID、导图 ID 或快照 | 阻止保存 |
| `MINDMAP_SNAPSHOT_INVALID` | mindmap | 导图内容格式异常，请重新打开后再试。 | 协议字段、根节点、编辑器标识异常 | 阻止写入 |
| `MINDMAP_SNAPSHOT_TOO_LARGE` | mindmap | 导图内容太大，暂时不能保存。 | 快照超过 5MB | 拆分内容或外置资产 |
| `MINDMAP_INLINE_ASSET_BLOCKED` | asset | 导图里有过大的内嵌素材，暂时不能保存。 | base64 data URL 超过阈值 | 进入资产模块后外置 |
| `DOCUMENT_REQUEST_INVALID` | document | 文档请求不完整，请重新打开后再试。 | 缺 `course_id + mind_map_id + node_id` | 阻止保存 |
| `DOCUMENT_NODE_MISSING` | document | 请先保存导图，再保存这个文档。 | 文档对应节点投影不存在 | 先保存导图 |
| `DOCUMENT_SNAPSHOT_INVALID` | document | 文档内容格式异常，请重新打开后再试。 | Word 快照协议异常 | 阻止写入 |
| `DOCUMENT_SNAPSHOT_TOO_LARGE` | document | 文档内容太大，暂时不能保存。 | 快照超过 2MB | 拆分内容或外置资产 |
| `DOCUMENT_INLINE_ASSET_BLOCKED` | asset | 文档里有过大的内嵌素材，暂时不能保存。 | base64 data URL 超过阈值 | 进入资产模块后外置 |
| `CHROME_NOT_FOUND` | chrome | 没有找到 Chrome，请检查浏览器路径。 | 未找到 Chrome 程序 | 配置路径 |
| `CHROME_PORT_NOT_CONFIGURED` | chrome | 浏览器端口还没配置好。 | 端口缺失或不可用 | 到端口管理配置 |
| `CHROME_LOGIN_REQUIRED` | chrome | 需要先确认浏览器登录状态。 | 未登录、需要验证 | 完成登录 |
| `AI_PROMPT_EMPTY` | ai | 请输入要发送给 AI 的内容。 | 提示词为空 | 前端提示补充 |
| `AI_PROVIDER_UNAVAILABLE` | ai | AI 服务暂时不可用，请稍后再试。 | 页面、端口、脚本执行失败 | 保留输入，允许重试 |
| `AI_NO_RESPONSE` | ai | AI 暂时没有返回结果，请稍后再试。 | 页面无可读取回答 | 记录阻塞原因 |
| `UPDATE_CONFIG_INVALID` | update | 更新配置不可用，请稍后再试。 | 仓库、Release、版本信息缺失 | 阻止更新流程 |
| `UPDATE_CHECK_FAILED` | update | 更新检测没有完成，请稍后再试。 | GitHub 查询失败 | 允许重试 |
| `UPDATE_DOWNLOAD_FAILED` | update | 安装包下载没有完成，请稍后再试。 | 下载地址、网络、写文件失败 | 允许重试 |
| `UPDATE_INSTALL_FAILED` | update | 安装程序没有启动，请稍后再试。 | 安装包不存在或启动失败 | 重新下载或检查文件 |
| `ERROR_LOG_READ_FAILED` | error-log | 报错日志暂时无法读取。 | 日志表不可用或查询失败 | 不阻断主流程 |
| `IMPORT_SOURCE_INVALID` | import | 导入文件暂时无法识别。 | 文件不存在、格式不支持、解析失败 | 导入器解析阶段使用 |
| `IMPORT_NODE_MATCH_FAILED` | import | 导入内容没有匹配到对应目录。 | 无法匹配稳定 `node_id` | 导入器匹配阶段使用 |
| `IMPORT_WRITE_FAILED` | import | 导入内容没有保存成功，请稍后再试。 | 导入快照写入失败 | 复用文档保存服务 |

## 接入规则

- 新模块必须优先使用已有错误码。
- 新错误码必须先进入 `electron/appErrors.ts`，再接业务代码。
- 新错误码必须补充本文件。
- 不允许在 UI 中直接显示 `technical_message`。
- 不允许把数据库错误、文件路径、堆栈传到前端弹窗。
- 导入器模块后续只负责产生结构化阶段错误，最终写入仍复用文档保存服务。
